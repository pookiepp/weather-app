Weather App

This is a simple weather application built using HTML, CSS, and JavaScript.
It lets you search for any city and shows the current temperature, weather condition, humidity, and wind speed.
The app uses the WeatherAPI.com free API to fetch live weather data.

Features

Search for weather by city name

Shows temperature, condition, humidity, and wind speed

Displays weather icons

Clean and minimal UI

Works on all screen sizes (responsive)

How to Use

Enter any city name in the search box

Click the Search button

The weather details will appear instantly

That’s it — no backend or extra setup needed.

Setup Instructions

Clone this repository:

git clone https://github.com/pookiepp/weather-app.git


Open the project folder

Open index.html in your browser

If the app doesn’t show data, add your own API key:

Get a free key from: https://www.weatherapi.com/

Replace your key in script.js:

const apiKey = "YOUR_API_KEY";
